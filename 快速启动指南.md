# 🚀 RAG智能问答系统 - 快速启动指南

## 📋 前置要求

- Python 3.8 或更高版本
- 网络连接（用于下载依赖和调用API）

## 🎯 方式一：使用硅基流动API（推荐，最简单快速）

### 步骤1：安装依赖


```bash
# 设置环境变量（使用你提供的硅基流动API密钥）
export USE_SILICONFLOW=true
export SILICONFLOW_API_KEY=sk-ziyyhopxmxuffgjgwphatkqjxftfenxvmlhpcycnzisdxyrr
```

> 💡 **提示**：如果想永久保存，可以将这些环境变量添加到 `~/.zshrc` 或 `~/.bash_profile` 文件中

### 步骤3：启动服务器

```bash
# 进入flask_app目录
cd flask_app

# 启动服务器
python3 run.py
```

### 步骤4：访问Web界面

打开浏览器，访问：**http://localhost:5050**

## 📝 使用流程

### 1. 注册/登录

- 首次访问会自动跳转到登录页面
- 点击"注册"标签创建新账户
- 输入用户名和密码完成注册
- 注册成功后会自动切换到登录标签

### 2. 上传文档

- 登录后进入主界面
- 点击左侧"上传文档"按钮
- 选择要上传的文件（支持 .txt、.html、.pdf 格式）
- 可以一次选择多个文件
- 上传成功后，文档会被处理并建立索引

### 3. 开始问答

- 在底部输入框输入你的问题
- 按 `Enter` 或点击发送按钮
- AI会基于你上传的文档内容回答
- 可以查看"参考文档"了解答案来源

### 4. 管理功能

- **新建对话**：清除当前对话历史
- **清空文档**：删除所有已上传的文档
- **退出登录**：安全退出账户

## 🔧 常见问题

### Q1: 端口被占用怎么办？

```bash
# 方法1：使用其他端口
export PORT=8080
cd flask_app
python3 run.py

# 方法2：关闭占用5050端口的程序
lsof -ti:5050 | xargs kill -9
```

### Q2: 依赖安装失败？

```bash
# 使用国内镜像源加速
pip3 install -i https://pypi.tuna.tsinghua.edu.cn/simple flask requests ...
```

### Q3: 硅基流动API调用失败？

- 检查API密钥是否正确
- 确认网络连接正常
- 查看控制台错误信息

### Q4: Embedding模型加载慢？

- 首次加载会下载模型，需要等待
- 如果本地模型不存在，会自动使用默认模型
- 可以通过环境变量指定模型路径
