## 接口文档

### 1. 登录接口

**路由:** `/login`  
**方法:** `POST`  
**描述:** 处理用户登录请求，验证用户名和密码。

**请求参数:**

- `username` (表单数据): 用户名
- `password` (表单数据): 密码

**响应:**

- 成功登录: 重定向到 `/index`
- 登录失败: 返回错误消息

**示例:**

```http
POST /login HTTP/1.1
Content-Type: application/x-www-form-urlencoded

username=user&password=pass
```

### 2. 注册接口

**路由:** `/register`  
**方法:** `POST`  
**描述:** 处理用户注册请求，创建新用户。

**请求参数:**

- `username` (表单数据): 用户名
- `password` (表形数据): 密码

**响应:**

- 注册成功: 重定向到 `/login`
- 注册失败: 返回错误消息

**示例:**

```http
POST /register HTTP/1.1
Content-Type: application/x-www-form-urlencoded

username=newuser&password=newpass
```

### 3. 登出接口

**路由:** `/logout`  
**方法:** `GET`  
**描述:** 处理用户登出请求，清除 session 中的用户名。

**响应:**

- 登出成功: 重定向到 `/login`

**示例:**

```http
GET /logout HTTP/1.1
```

### 4. 清空历史记录接口

**路由:** `/clear_history`  
**方法:** `POST`  
**描述:** 清空上传的文件路径列表和数据库连接。

**响应:**

- 成功: 返回 JSON 消息 `{'message': 'Files deleted successfully'}`
- 失败: 返回错误消息

**示例:**

```http
POST /clear_history HTTP/1.1
Content-Type: application/json

{}
```

### 5. 文件上传接口

**路由:** `/upload`  
**方法:** `POST`  
**描述:** 上传文件并更新数据库。

**请求参数:**

- `file` (文件上传): 上传的文件

**响应:**

- 成功: 返回上传文件的路径
- 失败: 返回错误消息

**示例:**

```http
POST /upload HTTP/1.1
Content-Type: multipart/form-data

--boundary
Content-Disposition: form-data; name="file"; filename="example.txt"

<文件内容>
--boundary--
```

### 6. 问答接口

**路由:** `/ask`  
**方法:** `POST`  
**描述:** 发送查询请求，获取模型的回答和检索结果。

**请求参数:**

- `query` (JSON 数据): 查询文本

**响应:**

- 成功: 返回 JSON 数据 `{'answer': answer, 'retrieved_result': retrieved_result}`
- 失败: 返回错误消息

**示例:**

```http
POST /ask HTTP/1.1
Content-Type: application/json

{
  "query": "什么是机器学习?"
}
```

### 7. 反馈接口

**路由:** `/feedback`  
**方法:** `POST`  
**描述:** 收集用户反馈并存储到文件中。

**请求参数:**

- `rating` (JSON 数据): 反馈评分
- `answer` (JSON 数据): 模型回答
- `retrieved_result` (JSON 数据): 检索结果文本列表

**响应:**

- 成功: 返回 JSON 消息 `{'message': 'Feedback received!'}`

**示例:**

```http
POST /feedback HTTP/1.1
Content-Type: application/json

{
  "rating": "good" | "bad" ,
  "answer": "机器学习是一种人工智能技术。",
  "retrieved_result": ["文本1", "文本2"]
}
```

### 8. 首页接口

**路由:** `/index`  
**方法:** `GET`  
**描述:** 返回首页模板，需要登录才能访问。

**响应:**

- 成功: 返回 `index.html` 模板
- 未登录: 重定向到 `/login`

**示例:**

```http
GET /index HTTP/1.1
```

### 权限控制

- 所有接口都需要用户登录才能访问，未登录用户会返回错误消息或重定向到登录页面。
- 登录状态通过 session 来维持。
