# RAG智能问答系统 - 启动说明

## 项目简介
这是一个基于RAG（检索增强生成）的智能问答系统，支持文档上传和基于文档内容的智能问答。

## 功能特性
- ✅ 用户注册/登录
- ✅ 文档上传（支持txt、html、pdf格式）
- ✅ 基于文档的智能问答
- ✅ 对话历史记录
- ✅ 美观的Web交互界面
- ✅ 支持多种大模型（本地模型/OpenAI API）

## 环境要求
- Python 3.8+
- 已安装项目依赖（requirements.txt）

## 快速启动

### 方式一：使用本地大模型（ChatGLM4）

1. **启动本地大模型服务**（如果使用本地模型）
   ```bash
   # 在llm_models目录下启动GLM4服务
   cd llm_models
   python glm4.py
   # 服务会在 http://localhost:8000 运行
   ```

2. **启动Flask应用**
   ```bash
   cd flask_app
   python run.py
   # 或者
   python3 run.py
   ```

3. **访问Web界面**
   - 打开浏览器访问：http://localhost:5050
   - 首次使用需要注册账户
   - 登录后可以上传文档并开始问答

### 方式二：使用硅基流动API（推荐，响应快且支持中文）

硅基流动提供高质量的中文大模型API服务：

1. **设置环境变量**
   ```bash
   export USE_SILICONFLOW=true
   export SILICONFLOW_API_KEY=your_siliconflow_api_key_here
   export SILICONFLOW_MODEL=Qwen/QwQ-32B  # 可选，默认Qwen/QwQ-32B
   ```

2. **启动Flask应用**
   ```bash
   cd flask_app
   python run.py
   ```

3. **访问Web界面**
   - 打开浏览器访问：http://localhost:5050

### 方式三：使用OpenAI API

如果本地模型响应慢，也可以使用OpenAI API：

1. **设置环境变量**
   ```bash
   export USE_OPENAI=true
   export OPENAI_API_KEY=your_openai_api_key_here
   export OPENAI_MODEL=gpt-3.5-turbo  # 可选，默认gpt-3.5-turbo
   ```

2. **启动Flask应用**
   ```bash
   cd flask_app
   python run.py
   ```

3. **访问Web界面**
   - 打开浏览器访问：http://localhost:5050

## 配置说明

### 环境变量配置

**模型选择（优先级从高到低）：**
- `USE_SILICONFLOW`: 设置为 `true` 使用硅基流动API（推荐）
- `SILICONFLOW_API_KEY`: 硅基流动API密钥（必需）
- `SILICONFLOW_MODEL`: 硅基流动模型名称（默认：Qwen/QwQ-32B）
- `SILICONFLOW_API_URL`: 硅基流动API地址（默认：https://api.siliconflow.cn/v1/chat/completions）

- `USE_OPENAI`: 设置为 `true` 使用OpenAI API
- `OPENAI_API_KEY`: OpenAI API密钥（使用OpenAI时必需）
- `OPENAI_MODEL`: OpenAI模型名称（默认：gpt-3.5-turbo）

- `LOCAL_LLM_URL`: 本地模型服务地址（默认：http://localhost:8000/chat/completions）

**其他配置：**
- `PORT`: Flask服务端口（默认：5050）
- `EMBEDDING_MODEL_PATH`:  embedding模型路径
- `EMBEDDING_DEVICE`: embedding模型设备（cpu/cuda）

### 使用示例

```bash
# 使用硅基流动API启动（推荐）
USE_SILICONFLOW=true SILICONFLOW_API_KEY=sk-xxx python flask_app/run.py

# 使用OpenAI API启动
USE_OPENAI=true OPENAI_API_KEY=sk-xxx python flask_app/run.py

# 使用本地模型，自定义端口
PORT=8080 python flask_app/run.py
```

## 使用流程

1. **注册/登录**
   - 首次访问会自动跳转到登录页面
   - 点击"注册"标签创建新账户
   - 登录后进入主界面

2. **上传文档**
   - 点击左侧"上传文档"按钮
   - 选择要上传的文件（支持txt、html、pdf）
   - 上传成功后文档会被处理并建立索引

3. **开始问答**
   - 在底部输入框输入问题
   - 按Enter或点击发送按钮
   - AI会基于上传的文档内容回答您的问题

4. **管理文档**
   - 点击"清空文档"可以删除所有已上传的文档
   - 点击"新建对话"可以清除对话历史

## 常见问题

### 1. 大模型响应慢怎么办？
- **推荐**：使用硅基流动API（设置USE_SILICONFLOW=true），响应快且支持中文
- 或使用OpenAI API（设置USE_OPENAI=true）
- 或使用更快的本地模型（如ChatGLM3）

### 2. 上传文件后无法回答问题？
- 检查文件格式是否支持（txt、html、pdf）
- 查看控制台是否有错误信息
- 确认embedding模型已正确加载

### 3. 端口被占用？
- 修改PORT环境变量：`PORT=8080 python flask_app/run.py`
- 或修改flask_app/run.py中的端口号

### 4. API调用失败？
- **硅基流动API**：
  - 检查API密钥是否正确
  - 确认账户有足够的余额
  - 检查网络连接（需要能访问api.siliconflow.cn）
- **OpenAI API**：
  - 检查API密钥是否正确
  - 确认账户有足够的余额
  - 检查网络连接
